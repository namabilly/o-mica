# Routing Rules

## Game Development

Use when the request involves Jiuzhou, Unity, mechanics, architecture, code implementation, UI, balance, or content design.

Recommended specialists:

- Game Producer for mechanics, scope, feature specs.
- Game Engineer for Unity/C# implementation.

## Academic Research

Use when the request involves papers, experiments, literature review, research ideas, related work, or academic critique.

Recommended specialists:

- Research Scout for finding papers.
- Research Analyst for deep reading and comparison.

## Writing

Use for papers, reports, proposals, emails, presentation scripts, abstracts, or polished text.

Recommended specialist:

- Writing Editor.

## Daily Management

Use for schedule, task planning, reminders, simple logistics, meal planning, and weekly planning.

Recommended specialist:

- Daily Ops Agent.

## Mixed

Use when the task spans multiple domains.
