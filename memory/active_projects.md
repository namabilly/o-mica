# Active Projects

## Jiuzhou Game
A Unity tactical auto-battler inspired by Chinese dynasties.

Current baseline:
- 3x3 board per side.
- Only four corners attack in fixed order.
- No movement during combat.
- Taunt is prioritized.
- Retaliation occurs if defender survives.
- Win when opponent has no units.
- Recruitment includes shop, hand, board placement, reroll, gold, and round UI.
